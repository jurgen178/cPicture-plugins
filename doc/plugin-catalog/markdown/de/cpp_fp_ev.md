### Belichtungsunterschied

Dieses Funktionszusatzmodul berechnet den Belichtungsunterschied (EV) aller ausgewählten Bilder relativ zum ersten ausgewählten Bild als Referenz. Das Ergebnis wird in einer Übersicht angezeigt.
