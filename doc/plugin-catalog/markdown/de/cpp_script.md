### Skript-Funktionszusatzmodul

Dieses Funktionszusatzmodul ermöglicht das Ausführen von PowerShell-, Batch- und Python-Skripten für die ausgewählten Bilder. Es wird automatisch installiert, wenn ein Skript-Plugin installiert wird.
