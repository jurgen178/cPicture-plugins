### Ascii Art

Dieses Funktionszusatzmodul konvertiert ein ausgewähltes Bild in ein Buchstabenbild (Ascii Art). Im Dialog können Schriftart, Blockgröße, Helligkeit und Kontrast angepasst werden. Das Ergebnis kann als Bild gespeichert werden.
